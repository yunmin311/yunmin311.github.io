# NOTICE — assets/ 里每个文件的版权与授权

> 下面每一行**版权声明**都是用 fontTools 从字体二进制自己的 `name` 表(nameID 0)里读出来的,
> 不是手打的 —— 重跑 `python gen_notice.py` 可复现。许可正文见同目录的 `OFL-1.1.txt`。

OFL 1.1 第 2 条要求:再分发时**每一份拷贝都必须随附版权声明与许可正文**
(*"provided that each copy contains the above copyright notice and this license"*)。
版权声明在字体自己的 `name` 表里(属于该条允许的 "machine-readable metadata fields");
**许可正文由本目录的 `OFL-1.1.txt` 以 "stand-alone text files" 形式满足** —— 
因为 Google Fonts 的网页字体和 pyftsubset 默认都会把 nameID 13(License Description)剥掉。

## 保留字体名(OFL 第 3 条)

第 3 条禁止 Modified Version 使用保留字体名(RFN),除非获得版权持有者明确书面许可。涉及两处:

- **LXGW WenKai**(RFN:`霞鹜` `霞鶩` `落霞孤鹜` `落霞孤鶩` `LXGW`)—— 我们自己做了 subset,
  属于 Modified Version。但作者在 OFL 里给了**附加许可**,原文明确覆盖本用法:
  *"…or in Modified Versions subsetted or converted to other formats (e.g., WOFF/WOFF2)
  **solely for web font delivery**, provided such Modified Versions are not made available as
  installable desktop fonts"*。我们交付的是 woff2 网页字体、不是可安装的桌面字体,合规。
  见 `LXGWWenKai-OFL.txt`。

- **其余 11 个**由 Google Fonts 服务端 subset 后分发,沿用原名 —— 
  这是自托管 Google Fonts 的通行做法,subset 由 Google 完成并以该名称对外分发。

## 明确排除(不在 assets/ 里,也不许加进来)

- **Klim Type Foundry 的 Untitled Sans / Untitled Serif** —— craft.do 自己用的字体,
  商业授权零售字体。颜色可以取,字体不行。
- **Fontshare(Gambetta / Zodiak / Erode / Sentient 等)** —— ITF Free Font License
  禁止再分发字体文件、禁止打包进分发物。免费可用 ≠ 可再分发。
- **transparenttextures.com 的纹理** —— 溯源 Subtle Patterns 为 **CC BY-SA 3.0**,
  传染性 ShareAlike,商用有雷。v4 已改用 CC0 的 ambientCG Paper001。
